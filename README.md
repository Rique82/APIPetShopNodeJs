7- configurar o index.js com o express
8- começar pelas rotas dentro do router, importando o express e usando uma função chamada routes, passando as rotas a essa função após.
9- configurar o banco de dados.
10- criar as funções do controller e service iguais, sempre criando uma classe primeiro do mesmo.
